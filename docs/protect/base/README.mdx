# Polkadot Base

The **Polkadot Base** is a collection of all security vulnerabilities not included in the top ten list, providing a broader perspective on potential security challenges within Polkadot SDK development. This resource is useful to increase your understanding of various risks, their potential impacts on Substrate-based blockchains, and strategies for mitigation or prevention. By familiarizing yourself with vulnerabilities, you contribute to enhancing the security and robustness of both Substrate and the broader Polkadot ecosystem.
